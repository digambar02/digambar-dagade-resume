# Linear Regression as Geometry

*Published: Jan 2025*

Linear regression is usually introduced through formulas,
but at its core it is a **geometric projection problem**.

## The Space We Work In

Suppose we have \( n \) observations and \( p \) predictors.

\[
X \in \mathbb{R}^{n \times p}
\]

Each column of \( X \) is a direction in space.
